## Installation

1. **Backend:**
   - Navigate to the `backend` directory.
   - Install dependencies: `npm install`.
   - Start the server: `npm run dev`.

2. **Dashboard:**
   - Navigate to the `dashboard` directory.
   - Install dependencies: `npm install`.
   - Start the frontend application: `npm start`.
